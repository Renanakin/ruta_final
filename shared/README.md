# Shared

Código, tipos, utilidades y recursos compartidos entre frontend, backend y Alchemist.

## ¿Qué debe hacer este equipo?
- Mantener utilidades, tipos y recursos que sean realmente compartidos.
- Documentar bien cada recurso para evitar dependencias innecesarias.

## ¿Qué NO debe hacer?
- No poner lógica de negocio específica de un área.
- No acoplar recursos que solo use un servicio.

## Skills recomendadas
- TypeScript/Javascript utilidades
- Documentación de código
