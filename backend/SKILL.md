# Skill

Este modulo backend debe operar como una API modular enfocada en negocio, datos y seguridad.

## Enfoque propuesto

- Node.js/Express avanzado
- Diseno y documentacion de APIs REST
- Seguridad y autenticacion
- Testing de endpoints
- Gestion de base de datos relacional o NoSQL

## Limites del modulo

- No construir UI
- No meter logica especifica del frontend
- No acoplar reglas de negocio al modulo `Alchemist`
