# Frontend

Aquí vive la aplicación web (React, Vite, etc.).

## ¿Qué debe hacer este equipo?
- Desarrollar la interfaz de usuario y experiencia visual.
- Consumir la API del backend y el servicio Alchemist vía HTTP/REST o WebSocket.
- Mantener el código desacoplado del backend y Alchemist (no lógica de negocio aquí).

## ¿Qué NO debe hacer?
- No implementar lógica de negocio ni acceso directo a la base de datos.
- No modificar el backend ni el servicio Alchemist desde aquí.

## Skills recomendadas
- Diseño UI/UX
- Accesibilidad web
- Consumo de APIs REST
- Manejo de estado en React
- Pruebas de componentes
